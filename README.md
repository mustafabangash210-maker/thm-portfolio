# 🛡️ TryHackMe Portfolio

Welcome to my cybersecurity learning portfolio. This repo tracks the TryHackMe rooms I've completed, with proof of completion and my own notes for each one.

> **Note:** Room solutions/walkthroughs are **intentionally not published** — this portfolio documents proof of learning and my personal takeaways, in line with [TryHackMe's Terms of Service](https://tryhackme.com/terms).

## 📊 Stats
- Rooms completed: _update after each publish_
- Paths: Web · Linux · Networking · Pentesting

## 📁 Structure
```
thm-portfolio/
├── README.md          # this file (portfolio index)
├── rooms/             # one folder per completed room
│   └── <room>/
│       ├── README.md  # room details + notes
│       └── card.html  # shareable completion card
└── assets/            # badges, banners
```

## 🚀 How it's managed
A small Python tool (in this repo's companion script) creates a completion card and README per room and keeps this index in sync. New rooms are added as I finish them.
